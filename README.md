# otdr-visualizer

Simulador OTDR interativo para fins de aprendizado, engenharia de processos e controle de qualidade de fibra óptica.

## O que você encontra

- Cenários OTDR realistas (emendas críticas, macrocurvaturas, conectores sujos, splitters e rompimentos).
- Alertas de aprendizagem com dicas rápidas durante a simulação.
- FAQ e links úteis sobre OTDR, padrões e boas práticas de telecom.

## Como usar

Abra o arquivo `index.html` no navegador para interagir com os parâmetros de manufatura, selecionar cenários e visualizar o traço OTDR em tempo real.
